## Setup and Installation

1. Clone the repository:
   ```bash
   git clone [your-repo-link]
   cd portfolio(MERN)
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open http://localhost:5173 to view the website
